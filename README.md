# How it works

> Every conduit related classes are in the modules folder
> - modules/conduit - common/base classes
> - modules/api - api related classes and services

# Getting started

```
docker-compose up
docker exec -it phpunit-mockery-workshop_web_1 /bin/bash

composer install
./yii migrate
```
